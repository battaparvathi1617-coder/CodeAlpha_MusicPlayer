# CodeAlpha Music Player

## 📌 Project Description
This is a responsive Music Player web application developed using HTML, CSS, and JavaScript as part of the CodeAlpha Frontend Development Internship. It provides a simple and interactive interface to play music with essential audio controls.

## 🚀 Features
- Play and Pause Music
- Next and Previous Track
- Song Title and Artist Display
- Progress Bar
- Volume Control
- Responsive Design
- Smooth Animations
- Playlist Support (Optional)
- Autoplay Feature (Optional)

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript

## 📂 Project Structure
```
CodeAlpha_MusicPlayer/
│── index.html
│── style.css
│── script.js
│── songs/
│── images/
│── README.md
```

## ▶️ How to Run
1. Clone or download this repository.
2. Open the project folder.
3. Open `index.html` in your web browser.
